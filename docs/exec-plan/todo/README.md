Place execution plans here.
Format: `<sequence>-<name>.md`, where `<name>` is descriptive kebab-case and `<sequence>` is four-digit zero-padded through `9999`, then unpadded at `10000+`.
For execution branches, the filename suffix must match the branch description: `feat/initial-setup` uses a file such as `0007-initial-setup.md`.
Assign the next available sequence number so active-file creation order stays visible without checking Git history.
When a plan is completed and the PR is ready, move the file to `../done/`.
